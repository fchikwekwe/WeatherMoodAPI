"In order to use this microservice, you can send your location via POST request to /weather to get the current weather for your location. You can send your mood and your city to /mood via post request to receive an object with your mood, city and the current weather."


WeatherMood Params:

'/weather' takes one parameter: 'city'.

Examples of valid cities include San Francisco, Portland and Atlanta.

The default city is San Francisco if no city parameter is provided.


'/mood' takes two parameters: 'mood' and 'city'.

Mood can be any valid string.

Examples of valid cities include San Francisco, Portland and Atlanta.

The default city is San Francisco if no city parameter is provided.
